# 11 - Gestión de Riesgos de Seguridad en Negocios Digitales

---

## Un Valor Estratégico, No Solo Cumplimiento

La gestión de riesgos de seguridad es **más que un cumplimiento legal: es un valor estratégico**. Permite conocer cómo es el modelo de negocio e identificar los activos de información, así como las posibles amenazas o vulnerabilidades que pueden afectarlo.

A partir de ese conocimiento se identifican los **controles** que ayudarán a gestionar dichas amenazas y vulnerabilidades.

---

## Técnicas de Prevención de Pérdida de Datos (DLP) según el GDPR

La implementación del **GDPR** obliga a incorporar técnicas de prevención de pérdida de datos, que permiten:

| Técnica | Objetivo |
|---------|----------|
| **Clasificar** | Identificar qué tipo de información se tiene y dónde y bajo qué condiciones se almacenan los datos personales |
| **Monitorear** | Vigilar el flujo de los datos y establecer controles de acceso y/o intercambio |
| **Borrar / Suprimir** | Eliminar aquella información que no se requiera o no se necesite |
| **Restringir** | Controlar de forma automática el acceso a datos personales por parte de terceros no autorizados |
| **Alertar** | Generar alertas automáticas y controles de seguridad que protejan los datos |
| **Restaurar** | Recuperar la disponibilidad y el acceso a los datos de forma rápida en caso de incidente físico o técnico |
